# 简历亮点

## 3-5 条核心亮点

### 1. MCP 协议实践
> 基于 Model Context Protocol (MCP) 构建了 10 个标准化 AI Agent 开发 Skill，通过 StdioTransport 与 Cline/Claude 原生集成，覆盖开发全生命周期。

### 2. Workflow Advisor 智能编排
> 实现 Workflow Advisor 引擎，根据自然语言需求自动识别工作流类型（feature/bugfix/refactor/research），推荐最优 Skill 编排顺序和参数传递策略。

### 3. 标准化 Skill 体系
> 设计统一 Skill 接口规范（场景/输入/输出/下一步），确保 10 个 Skill 可插拔、可组合、可独立演示。每个 Skill 包含即用型工具定义和 self-contained 处理逻辑。

### 4. 端到端 Demo
> 完成"用户注册功能"端到端演示：需求 → 工作流推荐 → 技术方案 → 编码实现 → 代码审查 → 完成验证，全流程在 Cline 中自动化执行。

### 5. 工程化思维
> 遵循 MVP 优先原则，避免 CrewAI/LangGraph 等外部依赖，纯 MCP SDK + Node.js 实现轻量级 Agent 系统，面试时可直接演示源码和运行效果。
