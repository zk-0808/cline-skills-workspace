核心设计原则:
  1. 每个 Handler = 一个独立 MCP 工具
  2. 统一输出格式: 方法论 + 结果 + Skill 元数据
  3. Workflow Advisor 负责任务编排，不负责执行
  4. YAML 定义工作流模板，Handler 实现具体逻辑
